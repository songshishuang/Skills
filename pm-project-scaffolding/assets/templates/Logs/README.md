# Logs/

Claude Code（或其它 AI 协作工具）的每日对话复盘归档。

## 结构

```
Logs/
├── YYYY-MM/
│   ├── YYYY-MM-DD.md         单日复盘
│   └── ...
├── YYYY-MM_汇总.md            月度汇总
└── ...
```

## 单日复盘必含

- 当日核心目标与产出
- 业务价值萃取（基建 / 护城河 / 数据等带来什么）
- 💡 Prompt 优化建议（反向给用户优化指令质量）
- 算力消耗追踪（按 session 估算 token）

## 月度汇总必含

- 全月核心主线
- 累积资产价值
- 提示词最佳实践演进
- 累计算力消耗

## 沉淀准则

- **每天工作结束触发归档** — 不要让对话淹没
- **跨日同 session 拆分归档** — 一个 session 跨多天就拆多份单日复盘
- **个人信息脱敏** — 涉及客户 / 同事的真名用化名

目前为空。今天的工作日志将创建在 `{{YEAR_MONTH}}/` 子目录下。
