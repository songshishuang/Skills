# 数据分析 (Data Analysis)

本目录沉淀 **{{PROJECT_NAME}}** 上线后的产品效果数据分析。

## 何时启用

产品 GA 之后——客户开始用、有埋点数据回流时。

## 推荐内容

- 关键功能的使用率 / 留存 / 转化漏斗分析
- A/B 实验结果
- 用户行为洞察报告
- 北极星指标的周 / 月 / 季度跟踪
- 业务专项分析（如流失原因 / 高潜识别 / 客单价分布）

## 命名约定

`{metric-area}-report-{YYYY-Qn}.md` 或 `{topic}-analysis-{YYYY-MM}.md`

例：`retention-report-2026-Q3.md` / `churn-analysis-2026-08.md`

## 与 `user-research/` 的区别

- 本目录回答 **"是什么"**（定量）：转化率 12.3% / 留存曲线下降 / 客单价中位数
- `user-research/` 回答 **"为什么"**（定性）：客户访谈中说"我不知道这个按钮在哪"

两者结合使用，结论才完整。
