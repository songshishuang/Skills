# 市场研究 (Market Research)

本目录沉淀 **{{PROJECT_NAME}}** 做市场决策时依赖的所有外部信息。

## 子目录约定

| 子目录 | 内容 | 何时新建 |
|---|---|---|
| `competitors/` | 竞品分析（产品架构图、功能拆解、定价策略） | 评估竞品时 |
| `industry-reports/` | 行业报告（艾瑞 / IDC / Gartner 等） | 撰写商业计划 / 立项时引用 |
| `trends/` | 行业趋势观察（监管动态、技术演进、客户支付意愿） | 季度产品规划复盘前 |

**只放原始资料与第三方分析**——自己的产品规划放 `../product-planning/`。
