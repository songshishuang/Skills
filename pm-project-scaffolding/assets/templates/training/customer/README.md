## 写作准则

- **不要写「产品有什么功能」，要写「客户怎么完成任务」**
- **每个任务 ≤ 5 步**——超过说明任务太复杂或描述太啰嗦
- **图比文重要**
- **保持版本号**——每篇顶部标注「适用版本 vX.Y」
