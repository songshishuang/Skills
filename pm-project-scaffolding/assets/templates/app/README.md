# app/

**占位目录**：用于未来真正的代码开发实现。

当前 [`../prototypes/`](../prototypes/) 下是高保真原型（无构建链、单文件渲染）。当 PRD 进入实际工程交付阶段时，前端 / 后端代码（如 React/Vue 工程、API 服务等）落在本目录下。

## 何时启用

- PRD 评审通过
- 原型阶段验证完关键设计决策
- 团队决定进入工程实施

## 设计哲学

AI Native 架构最大的风险是过早工程化被沉没成本绑架。先用原型阶段把交互模式、数据结构、能力边界这些**最贵的设计决策**先验证出来，再启动工程能把返工成本压到最低。
